# PlaylistOrganizer

**PlaylistOrganizer** est une application web conçue pour aider les utilisateurs à organiser et gérer efficacement leurs playlists musicales.

## Fonctionnalités

- **Gestion des playlists** : Créez, modifiez et supprimez des playlists selon vos préférences.
- **Affichage dynamique** : Visualisez vos playlists avec une interface utilisateur intuitive et réactive.
- **Pagination** : Parcourez vos playlists facilement grâce à une pagination intégrée.

## Prérequis

- **Node.js** : Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé sur votre machine.

## Installation

1. **Cloner le dépôt** :

   ```bash
   git clone https://github.com/yazstonn/PlaylistOrganizer.git
   ```

2. **Accéder au répertoire du projet** :

   ```bash
   cd PlaylistOrganizer
   ```

3. **Installer les dépendances** :

   ```bash
   npm install
   ```

## Utilisation

1. **Démarrer l'application** :

   ```bash
   npm start
   ```

2. **Accéder à l'application** : Ouvrez votre navigateur et rendez-vous sur `http://localhost:3000` pour commencer à utiliser PlaylistOrganizer.

## Structure du Projet

- **`app.js`** : Point d'entrée principal de l'application.
- **`routes/`** : Contient les définitions des routes de l'application.
- **`views/`** : Contient les templates Pug pour le rendu côté serveur.
- **`public/`** : Contient les fichiers statiques tels que les styles CSS et les scripts JavaScript.
